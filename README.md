# Projeto-Calculadora

Uma calculadora simples desenvolvida para praticar lógica de programação, manipulação do DOM e eventos com JavaScript.

## Link da aplicação

Acesse aqui: https://projet-calculadora.netlify.app/

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Operações básicas (+, -, ×, ÷)
- Limpar visor
- Apagar último número
- Interface responsiva
- Cálculos em tempo real

## Estrutura do projeto

```
calculator/

│── index.html
│── style.css
│── script.js
│── README.md
```

## Aprendizados

- Manipulação do DOM
- Eventos com JavaScript
- Operações matemáticas
- Organização de código
- Estruturação de interface
